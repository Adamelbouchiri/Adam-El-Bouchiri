## Featured Work

### Production Work

#### Taahud — Construction SaaS Platform
A multi-tenant marketplace connecting customers, engineers, suppliers, and developers across three project arenas. Multi-account-type authentication with OTP verification (Sanctum), Moyasar subscription billing with plan-based posting limits, role-based access control per arena, applications/bids system, file uploads, and an admin layer. Fully Arabic, RTL.
<sub>Laravel · React · Tailwind · Sanctum · Moyasar · MySQL  &nbsp;·&nbsp;  *Private — Taahud-Platform-Backend, Taahud-App*</sub>

#### Trading App — Crypto Trading Dashboard
A live crypto trading interface with buy/sell flows, market data, user wallets, Stripe-powered checkout, and a full admin panel for managing users and transactions. Deployed in production.
<sub>Laravel · React · Vite · Stripe · MySQL  &nbsp;·&nbsp;  *Private — Trading-app-front-end, Trading-app-back-end*</sub>

### Open Source

#### Dentist — Dental Clinic Booking Platform
A full-stack appointment booking system. Patients sign up via email or Google/Facebook OAuth, complete a two-step profile, book appointments, and pay through Stripe. Admin dashboard for managing appointments and users, with full settings (avatar, name, password updates) and password reset flow.
<sub>React · Laravel · Sanctum · Stripe · Tailwind · Framer Motion  &nbsp;·&nbsp;  </sub>[**Frontend →**](https://github.com/Adamelbouchiri/Dentist) &nbsp;·&nbsp; [**Backend →**](https://github.com/Adamelbouchiri/Dentist-backend)

#### Laravel E-Learning Platform
A three-role learning management system (admin · coach · student). Coaches create classes, courses, and lessons with auto-graded project submissions. Students enroll in paid classes, progress through lessons sequentially, and track their schedule on a calendar view. Admins approve users and promote them to coach.
<sub>Laravel · Blade · MySQL · Tailwind  &nbsp;·&nbsp;  </sub>[**View repo →**](https://github.com/Adamelbouchiri/laravel-final-project)

#### Laravel E-Commerce Store
A role-based e-commerce platform (admin · moderator · customer) with product management, shopping cart with quantity controls, checkout flow, and soft-delete user management with restore-from-trash.
<sub>Laravel · Blade · MySQL · Tailwind  &nbsp;·&nbsp;  </sub>[**View repo →**](https://github.com/Adamelbouchiri/laravel-store-with-advanced-stuff)

#### 3D Portfolio
An interactive personal portfolio with 3D scenes built on React Three Fiber and Three.js. Vertical timeline for experience, scroll-triggered animations via Framer Motion, and a working contact form powered by EmailJS.
<sub>React · Three.js · React Three Fiber · Framer Motion · EmailJS  &nbsp;·&nbsp;  </sub>[**View repo →**](https://github.com/Adamelbouchiri/3D-Portfolio-React.js)
